
# @cougargrades/atom-feed

Port of [@au5ton/use-atom-feed](https://github.com/au5ton/use-atom-feed) that only parses and validates [Atom feeds](https://en.wikipedia.org/wiki/Atom_(web_standard)) into JS objects (No longer specific to React). Depends on [DOMParser](https://developer.mozilla.org/en-US/docs/Web/API/DOMParser) being globally available, which is typically only available in browser runtimes.
